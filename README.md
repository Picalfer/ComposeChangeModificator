# Compose Change Modificator

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![Jetpack Compose](https://img.shields.io/badge/Jetpack-Compose-blue.svg?logo=android)](https://developer.android.com/jetpack/compose)

Тестовое приложение для изучения модификаторов и состояний в Jetpack Compose.

## О проекте

Приложение демонстрирует работу с модификаторами Compose и управление состоянием интерфейса. Показывает onboarding поверх основного контента с измененными свойствами элементов.

**Основной функционал:**
- Управление видимостью onboarding
- Изменение модификаторов элементов (прозрачность, границы)
- Наложение элементов с затемнением фона
- Работа с состоянием в Compose

## Технологии

* Kotlin
* Jetpack Compose
* Material 3
* State management
